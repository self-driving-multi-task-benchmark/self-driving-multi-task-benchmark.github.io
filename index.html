Self Driving Multi Task Benchmark

Multi-task learning jointly training shared parameters on multiple tasks, mining latent information among these tasks to improve efficiency and accuracy.

Multi-task learning architectures apply parameter sharing to learn share information between different tasks. According to the sharing methods, these architectures are divided into encoder-focused architectures and decoder-focused architectures. In encoder-focused architectures, parameters are only shared in encoder stage which can further divided into hard and soft parameter sharing, followed by task-specific heads.

We follow the common multi-task architecture where each task shares the same encoder and has its own specific head. The benchmark focuses on three types of task scheduling, four task balancing methods and two partial-label learning techniques and experiments on three common data split settings. Our code will be available at github.

Data Split

BDD100k dataset contains multiple tasks. Here we focus on object detection (OD), semantic segmentation (SS), drivable area segmentation (DA) and lane detection (LD). In BDD100K, 70k training images are labeled for object detection, drivable area segmentation and lane detection, and only 7k training images are labeled for semantic segmentation.

Disjoint-normal setting With the same annotation effort or budget, the quantity of annotations decreases with the increase of the complexity of annotating a task. Therefore, we consider a realistic scenario where each input image is labeled to only one task in this setting, namely, the annotations for each task are disjoint. The number of labelled images for each task is as follows: object detection (10k), semantic segmentation (7k), drivable area segmentation (20k), and lane detection (20k).

Disjoint-balance setting We further consider a more difficult setting that corresponds to the scenarios of scarce annotations. There are 21k images in this setting and each task has 7k labeled images that are not overlapped with other tasks.

nuImages dataset contains 73k images collected from diverse spatiotemporal distribution and weather and lighting conditions. All images are annotated in object detection (OD) and semantic segmentation (SS).
