    Multi Task Learning

    Multi-task learning jointly training shared parameters on multiple tasks, mining latent information among these tasks to improve efficiency and accuracy.

    Multi-task learning architectures apply parameter sharing to learn share information between different tasks. According to the sharing methods, these architectures are divided into encoder-focused architectures and decoder-focused architectures. In encoder-focused architectures, parameters are only shared in encoder stage which can further divided into hard and soft parameter sharing, followed by task-specific heads.

    Hard parameter sharing

    In hard parameter sharing, the shared part of the parameters are consistent throughout different tasks typically being the encoding part

    Soft parameter sharing

    In soft parameter sharing, each tasks has its own parameter setting and sharing is accomplished by feature sharing mechanism

    Decoder-focused

    For decoder-focused architectures, parameters are shard both in encoder and decoder stage.
